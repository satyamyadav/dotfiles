### dotfiles

> Managing dotfiles with stow for linux

I am using awesome wm and urxvt on Ubuntu 20.x 


#### Install

```bash

git clone https://github.com/satyamyadav/dotfiles.git .dotfiles

cd .dotfiles

stow bash git vim # name of configs
```
